About This App

SkyCast is a modern weather platform that delivers real-time weather updates for cities around the world.

Users can quickly check temperature, humidity, and wind conditions through a clean and interactive interface.

Built using HTML, CSS, Bootstrap, and JavaScript with live weather integration powered by the Open-Meteo API.

Usage Guide

1. Enter City

Type the name of any city in the search box.

2. Click Search

The app fetches live weather data using the Open-Meteo API.

3. View Weather

Check temperature, humidity, wind speed, sunrise, and sunset.
