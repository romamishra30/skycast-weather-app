# 🌤️ SkyCast Weather App

SkyCast is a modern and responsive weather application that provides real-time weather updates for cities around the world.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- 💧 Humidity information
- 🌬️ Wind speed details
- 📊 Weather table for common cities
- 🎨 Clean and responsive UI using Bootstrap
- ⚡ Dynamic weather data using Open-Meteo API

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Open-Meteo API

## 📸 Preview

SkyCast provides a clean dashboard with weather cards, city search, and additional weather details.

## 🌐 API Used

Open-Meteo Weather API

Example:

https://api.open-meteo.com/
